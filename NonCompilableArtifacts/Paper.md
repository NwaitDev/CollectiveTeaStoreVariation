# Designing with Chips, Executing with BIP: Applications going from modular robots to server monitoring

## Abstract

This paper presents the compilation chain currently developed to design applications with the Chips description language and turn these higher level specifications into component based BIP implementations, thus benefiting from the correct-by-construction c++ code generation it provided. Along with the toolchain, an extension inspired by aggregate programming is proposed for the Chips language. This additional feature allows to precisely coordinate large ensembles of embedded processes while still letting the user tune the behavior of the system according to control theory principles. Through the simulation of two example applications, namely an adaptive light implemented by a track of modular robots and an automatic load balancing web server, we demonstrate here the ease with which complex models of various systems can be handled thank to Chips. We present the compilation artifacts involved in the generation of the executable C++ code used to simulate these applications and show the results obtained by fine tuning controllers integrated to make the systems more resilient to environmental disturbances.
